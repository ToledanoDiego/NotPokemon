Not Pokemon

The game file is MyGame.py and is built in python3. To run the game copy the contents of the game file into codeskulptor3. There may be loading errors that cause the player to have to run the code again.
The game functions on other python clients however there may be loading errors such as missing sprites.
The source code for the game can also be accessed at https://pastebin.com/XPGQKE7N 


Not Pokemon is a project for CS1830 by the group Coconut.
